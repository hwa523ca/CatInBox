Hello! This program is a game called "Cat In The Box".
This game is based off of an interview question for Microsoft.
The premise of this game is to find the cat in a series of boxes.
The cat hides in one box, and if the player does not find it, the 
cat moves to another box adjacent from the one it previously hid in.
The game continues until the player finds the cat.
